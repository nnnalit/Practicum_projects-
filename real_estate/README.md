# Анализ рынка недвижимости Санкт-Петербурга и Ленинградской области

## 📌 Описание проекта

Анализ объявлений о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области.  
Цель — помочь агентству недвижимости определить самые перспективные сегменты рынка и спланировать бизнес-стратегию.

---

## 🎯 Задачи проекта

### Часть 1. Знакомство с данными
- Определить временной интервал данных (2014–2019)
- Изучить распределение объявлений по типам населённых пунктов
- Проанализировать время активности объявлений
- Рассчитать долю снятых с публикации объявлений
- Оценить соотношение объявлений Санкт-Петербурга и Ленобласти
- Проверить корректность данных (стоимость кв.м, площадь, комнаты, балконы, высота потолков)

### Часть 2. Решение ad hoc задач (SQL)

**Задача 1. Время активности объявлений**
- Разделить объявления на категории по длительности активности (1–30, 31–90, 91–180, 181+ дней, non category)
- Проанализировать количество, стоимость, площадь, комнаты и балконы для каждого региона
- Сравнить рынки Санкт-Петербурга и Ленинградской области

**Задача 2. Сезонность объявлений**
- Определить месяцы с наибольшей активностью публикаций и снятий
- Проанализировать влияние сезонных колебаний на стоимость кв.м и площадь
- Сформулировать рекомендации для маркетинговых кампаний

### Часть 3. Доработка дашборда в DataLens

**Вкладка «Аналитика рынка недвижимости»:**
- Добавлен фильтр по типам населённых пунктов
- Добавлены метрики: «Средняя площадь», «Среднее количество дней продажи»
- Создан чарт «Топ-5 по количеству дней продажи»
- Добавлен фильтр гранулярности (день, неделя, месяц, год)

**Вкладка «Детализация рынка»:**
- Добавлены названия населённых пунктов
- Сортировка по средней стоимости с цветовым градиентом
- Заменены круговые диаграммы на столбчатые (планировка, тип жилья)
- Добавлены диаграммы влияния водоёмов и парков на стоимость
- Создан график активности по дням недели с цветовым выделением
- Построен график сезонности по годам
