# Бизнес-метрики сервиса доставки еды «Всё.из.кафе»

## 📌 Описание проекта

Расчёт и визуализация ключевых бизнес-метрик сервиса доставки еды «Всё.из.кафе» в городе Саранске за период май — июнь 2021 года.  
Цель — разработать дашборд для анализа состояния клиентской базы и подготовить аналитическую записку с выводами.

---

## 🎯 Задачи проекта

### 1. Расчёт метрик (SQL + DataLens)
- **DAU** (Daily Active Users) — количество активных пользователей за день
- **Conversion Rate** — коэффициент конверсии (посещение → заказ)
- **Средний чек** — средняя сумма покупки на пользователя
- **LTV** (Lifetime Value) — совокупная ценность клиента за период
- **Retention Rate** — коэффициент удержания пользователей (когортный анализ)

### 2. Визуализация в DataLens
- Создание графиков для каждой метрики (линейные тренды, столбчатые диаграммы)
- Обеспечение возможности выявления закономерностей и аномалий

### 3. Создание дашборда
- Дашборд «Ключевые бизнес-метрики в Саранске (май — июнь 2021)»
- Добавление аналитической записки отдельной вкладкой

### 4. Аналитическая записка
- Анализ динамики DAU, Conversion Rate, среднего чека, Retention Rate
- Интерпретация трендов, выявление причин отклонений
- Рекомендации для бизнеса

---

## 📊 Результаты

- **DAU**: максимальное значение — 85 (4 мая)
- **Conversion Rate**: минимальное значение — 18% (18 июня)
- **Retention Rate** на 7-й день: 4% (майская когорта — 5%, июньская — 3%)
- **Средний чек**: май — 135.88 руб., июнь — 147.66 руб. (выше в июне)
- **Топ-1 ресторан по LTV**: «Гурманское Наслаждение» — 170 479 руб.
- **Топ-5 популярных блюд**: все содержат мясо
