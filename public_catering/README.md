# Исследование рынка общественного питания Москвы

## 📌 Описание проекта

Исследовательский анализ данных о заведениях общественного питания Москвы.  
Цель — помочь инвесторам выбрать концепцию, расположение и ценовой сегмент для открытия нового заведения.

---

## 🎯 Задачи проекта

### Шаг 1. Загрузка и знакомство с данными
- Загрузить данные из двух файлов: `rest_info.csv` и `rest_price.csv`
- Изучить структуру, объём, типы данных и пропуски
- Объединить датасеты в один для дальнейшей работы

### Шаг 2. Предобработка данных
- Преобразовать типы данных при необходимости
- Обработать пропущенные значения
- Проверить и устранить явные и неявные дубликаты (нормализация названий)
- Создать новый столбец `is_24_7` — признак круглосуточной работы

### Шаг 3. Исследовательский анализ данных (EDA)
- **Категории заведений**: распределение по типам (кафе, ресторан, бар, кофейня и др.)
- **Административные районы**: распределение заведений по районам Москвы, отдельно по ЦАО
- **Сетевые vs несетевые**: соотношение, какие категории чаще являются сетевыми
- **Посадочные места**: анализ распределения, выбросы, типичные значения по категориям
- **Рейтинги**: распределение средних рейтингов по категориям
- **Корреляционный анализ**: связь рейтинга с категорией, районом, сетевостью, количеством мест, ценой, круглосуточной работой
- **Топ-15 популярных сетей**: по количеству заведений, их рейтинги и категории
- **Анализ цен**: вариация среднего чека в зависимости от района, сравнение ЦАО и других районов

### Шаг 4. Итоговые выводы и рекомендации
- Обобщить результаты исследования
- Сформулировать рекомендации для инвесторов по выбору:
  - Типа заведения
  - Района расположения
  - Ценового сегмента
