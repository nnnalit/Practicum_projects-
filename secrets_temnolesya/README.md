# Аналитический проект «Секреты Темнолесья»

## 📌 Описание проекта

Анализ данных игрового проекта «Секреты Темнолесья».  
Исследование поведения игроков, внутриигровых покупок и проверка гипотез для маркетинговой команды.

---

## 🎯 Задачи проекта

### Часть 1. Исследовательский анализ данных

**Задача 1. Исследование доли платящих игроков**
- Рассчитать долю платящих игроков от общего числа зарегистрированных игроков
- Проанализировать зависимость доли платящих от расы персонажа

**Задача 2. Исследование внутриигровых покупок**
- Получить статистические показатели по стоимости покупок (количество, сумма, min, max, среднее, медиана, стандартное отклонение)
- Выявить покупки с нулевой стоимостью, определить их количество и долю
- Определить популярность эпических предметов (абсолютное и относительное количество продаж, доля покупателей)

### Часть 2. Решение ad hoc задач
- Проанализировать зависимость активности игроков от расы персонажа
- Проверить гипотезу о различиях в прохождении игры между расами

### Часть 3. Выводы и рекомендации
- Сформулировать выводы по результатам анализа
- Подготовить рекомендации для маркетинговой команды
