# Дашборд TED-конференций

## 📌 Описание проекта

Разработка дашборда в Yandex DataLens для анализа данных о конференциях TED.  
Цель — помочь компании, получившей лицензию на проведение TED-конференций, изучить опыт прошлых мероприятий и принять решения по организации первого события.

---

## 🎯 Задачи проекта

### Бизнес-вопросы заказчика:
- Какие темы популярны? Как зрители реагируют на них?
- Какое среднее количество выступающих на одной конференции?
- Какая средняя длина выступления? Как правильно организовать время?
- Кто из выступающих интересен зрителям?

### Наполнение дашборда:

**1. Верхнеуровневые показатели:**
- Количество конференций
- Количество выступлений
- Уникальное количество выступающих
- Количество тем / тегов

**2. Детализация по конференциям:**
- Фильтр по стране → детализация до конференций
- Средняя длительность выступления
- Максимальное количество просмотров у выступления
- Количество аплодисментов за всю конференцию
- Количество выступающих

**3. Исследование выступлений:**
- Топ-10 самых смешных выступлений (по laughter_count)
- Топ-20 самых популярных тегов

**4. Информация о выступающих:**
- Род деятельности выступающих, которым аплодируют больше всего

**5. Детальная информация:**
- Возможность просмотра информации о каждом выступлении

**6. Селекторы:**
- По стране
- По названию конференции
- По тегу выступления
- По дате записи

## 📊 Результаты

- Разработан дашборд в Yandex DataLens, решающий поставленные бизнес-задачи
- Настроены связи между чартами и селекторы для фильтрации данных
- Обеспечена возможность детализации от страны до конкретного выступления
- Визуализации оформлены в единой цветовой схеме с заголовками и легендами
