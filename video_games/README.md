# Рынок видеоигр

## 📌 Описание проекта

Исследование развития игровой индустрии с 2000 по 2013 год.  
Подготовка данных для статьи о развитии игровых платформ, жанров и региональных предпочтений игроков.

---

## 🎯 Задачи проекта

1. **Загрузка и знакомство с данными**
   - Изучить структуру данных, типы, пропуски
   - Сделать выводы о качестве данных

2. **Предобработка данных**
   - Привести названия столбцов к стилю `snake_case`
   - Преобразовать типы данных (числовые столбцы из `object` в `float64`)
   - Обработать пропуски:
     - удалить строки с пропусками в `name` и `genre`
     - удалить строки с пропусками в `year_of_release`
     - заменить пропуски в продажах на `0`
     - заменить пропуски в `rating` на `'UNKNOWN'`
   - Устранить неявные дубликаты (нормализация регистра)
   - Удалить явные дубликаты

3. **Фильтрация данных**
   - Отобрать данные за период 2000–2013 годы

4. **Категоризация данных**
   - Разделить игры по оценкам пользователей (высокая, средняя, низкая, не оценено)
   - Разделить игры по оценкам критиков (высокая, средняя, низкая, не оценено)
   - Выделить топ-7 платформ по количеству выпущенных игр

5. **Итоговый вывод**
   - Обобщить результаты подготовки данных

---

## 📊 Результаты

- Подготовлен датасет `df_actual` с 12 781 записью за 2000–2013 годы
- Созданы новые поля: `user_score_category`, `critic_score_category`
- Выявлены топ-7 платформ: PS2, DS, WII, PSP, X360, PS3, GBA
- Данные готовы для дальнейшего анализа
