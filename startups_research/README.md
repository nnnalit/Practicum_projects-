# Исследование стартапов

## 📌 Описание проекта

Анализ данных о финансировании стартапов: объёмы инвестиций, типы финансирования, динамика по годам, возвраты средств.  
Цель — помочь венчурной компании понять закономерности финансирования и оценить перспективы инвестирования.

---

## 🎯 Задачи проекта

### Шаг 1. Знакомство с данными и предобработка
- Загрузка данных из двух датасетов: `cb_investments.csv` и `cb_returns.csv`
- Приведение названий столбцов к единому стилю
- Преобразование типов данных (числовые, даты)
- Обработка пропусков и дубликатов
- Заполнение пропусков в `mid_funding_at` на основе `first_funding_at` и `last_funding_at`

### Шаг 2. Инжиниринг признаков
- Разделение компаний на группы по срокам финансирования:
  - Единичное финансирование
  - Срок финансирования до года
  - Срок финансирования более года
- Выделение сегментов рынка: массовые (>120 компаний), средние (35–120), нишевые (<35)
- Замена средних и нишевых сегментов на заглушки `mid` и `niche`

### Шаг 3. Работа с выбросами и анализ
- Анализ типичных и аномальных значений финансирования (метод IQR)
- Определение сегментов с наибольшей долей аномалий
- Фильтрация данных: проверка полноты за 2014 год, исключение аномалий
- Анализ типов финансирования по объёму и популярности
- Построение графика возвратов по типам финансирования

### Шаг 4. Анализ динамики
- Динамика среднего объёма раунда финансирования по годам
- Динамика количества раундов по годам
- Динамика размера общего финансирования по массовым сегментам (растущие в 2014 году)
- Годовая динамика доли возвращённых средств по типам финансирования (venture, debt_financing, private_equity, seed, angel)

### Шаг 5. Итоговые выводы и рекомендации
- Рекомендации по отраслям для инвестирования
- Рекомендации по типам финансирования
- Обобщение результатов анализа

---

## 📊 Результаты

- Проведена полная предобработка данных
- Выявлены сегменты рынка с наибольшим потенциалом
- Проанализирована динамика финансирования по годам
- Рассчитаны нормированные возвраты по типам финансирования
- Сформулированы рекомендации для инвестора на 2015 год
